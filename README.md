서브 컬쳐 게시판 버전1<br>
이 프로젝트는 서브 컬쳐를 주제로 다양한 이야기를 나누는 게시판 프로토타입 입니다.

프로젝트 진행 기간 : 2025-02-01 ~ 2025-03-09 <br>
개인 프로젝트 : 김찬중

주요 기능  <br>
서브 컬쳐, 게시판, 친구 신청 및 메시지 보내기
